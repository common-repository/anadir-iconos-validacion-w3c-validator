=== Widget para añadir los iconos de validación de W3C Validator ===Contributors: MavkSoft.esDonate link: https://www.mavksoft.es/Tags: widget, xhtml, html5, texto, visitantes, mavksoft, w3c, validator, muchomasqueunblog, pluginRequires at least: 1.0Tested up to: 4.4.5Stable tag: trunkLicense: GPLv2 or laterLicense URI: http://www.gnu.org/licenses/gpl-2.0.htmlEste Plugin nos muestra en cualquier lugar que se pueda añadir un Widget de texto los distintivos de validación de W3C Validator.== Description ==Con este plugin podemos dar el servicio a nuestros visitantes de ver que tenemos validada nuestra página con W3C Validator, y conseguir de esta forma aumentar nuestras visitas.== Installation ==1. Subir el archivo a wp-content/plugin2. Activar plugin3. Ir a Apariencia > widgets4. Arrastra el widget al sidebar que quieras.5. Comprueba que los distintivos se ven correctamente!
